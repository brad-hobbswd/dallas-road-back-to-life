# DALLAS and the Road Back to Life

A journey through darkness toward freedom.

Dallas and the Road Back to Life is an online book experience by Bradley Hobbs that follows the story of one man's journey through addiction, fear, temptation, surrender, and restoration. Through valleys, rivers, mountains, and moments of grace, Dallas discovers that true freedom comes through Jesus.

This project was designed as an immersive GitHub Pages reading experience where each section of the book unfolds as its own page, creating a steady reading flow from beginning to conclusion.
